Microsoft security patches

13 Oct, 2004 - 1 minutes
There’s some pretty serious security bugs fixed by a bunch of patches Microsoft released today. Most of them don’t apply to XP users who’ve applied Service Pack 2, but one of the more serious ones does. Service Pack 2 is a pretty large improvement in security for Windows, so if you haven’t installed it yet, you really oughta. The scariest of the flaws is MS04-038, which applies to Internet Explorer and affects everyone who uses it.
